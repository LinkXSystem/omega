# Omega 模块

## 模块结构

```shell
.
├── auxiliary  # 辅助模块，实现锚点功能
├── comment    # 注释模块,基于 node 模块实现
├── common     # 通用模块
├── connector  # 连接模块
├── coordinate # 坐标系统
├── grid       # 网格模块
├── line       # 线段模块
├── monitor    # 监控模块
├── node       # 节点模块（container），核心，围绕此拓展功能点
├── renderer   # 渲染模块
└── workspace  # 工作空间
```
